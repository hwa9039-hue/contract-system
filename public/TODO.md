기존 기능은 유지하면서 아래 요구사항을 반영하는 것이 목표입니다.

[요구사항]

1. 대시보드
- 총괄 그래프 삭제
- 공사, 민간 항목 제거
- 총 건수 / 총 계약금액 삭제
- 하단 년도별 정보는 유지

2. 버튼
- 삭제 버튼 너비를 저장/취소 버튼과 동일하게 수정

3. 계약현황 필터
- "사업년도 0000" → 실제 연도 숫자로 표시 (예: 2026)
- "준공일자 기준 D-Day 자동 표시" 문구 제거
- 해당 위치에 합계 금액 표시

4. 캘린더
- 일정 클릭 시 계약현황 상세 팝업과 동일한 UI로 표시

[작업 조건]
- App.jsx / App.css 중심 수정
- 기존 구조 유지
- 불필요한 코드 제거

[결과]
